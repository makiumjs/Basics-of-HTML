Basics of HTML

1.Document Outline
  • !DOCTYPE html> Version of HTML
  • html> Type of document (HTML Document in this case)
  • head> Page information
  • body> Page contents

2.Page information
  • base/> Base URL
  • meta/> Meta Data
  • title/> Title
  • link/> Relevante Resource
  • style/> Style Resource
  • script> Script Resource

3.Document Structure
  • h[1-6]> Heading
  • div> Page Section
  • span> Inline Section
  • p> Paragraph
  • br> Line Break
  • hr/> Horizontal Rule

4.Links
  • a href=""> Page link
  • a href="mailto:" Email link

  • a name="name" anchor

5.Lists

  • dd> term description
